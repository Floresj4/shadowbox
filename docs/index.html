shadowbox.js

A Javascript library providing simple shadow boxing functionality. shadowbox.js requires jquery and a valid selector – the content to be shadowed should exist on the page with a default css display value of none.

  • to avoid shadowbox content from flashing away on page load, style attributes for each item to be displayed should be set display:none

Usage


<script src="https://code.jquery.com/jquery-3.3.1.slim.min.js"></script>
<script type='text/javascript' src="./shadowbox.js"></script>
<script type='text/javascript'>
$(function(){
	shadowbox({
		background: 'url(./tile.png)'
		opacity: .7,
	});
});
</script>
<body>
	<div class='peekaboo-container'
		 shadowbox="open-me-too">
		<h4>Peekaboo</h4>
	</div>
</body>
