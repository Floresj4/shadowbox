shadowbox-jf.js

A Javascript library providing simple shadow boxing functionality. shadowbox-jf requires jquery and a valid selector – the content to be shadowed should exist on the page with a default css display value of none.

Installation

shadowbox-jf is available via Node Package Manager (npm) 

npm install shadowbox-jr --save
or downloading the js file from github directly.

Usage

shadowbox-jf uses an attribute placed on the container intended to diplay with a value corresponding to the selector to used to trigger the display event. In the example below, shadowbox='open-me' placed on the teapot-container indicates that it will have shadowbox functionality attached to it and an element containing open-me will be used to interact with the container.

Notes
  • To avoid shadowbox content from flashing away on page load, style attributes for each item to be displayed should be set display:none

<script src="https://code.jquery.com/jquery-3.3.1.slim.min.js"></script>
<script type='text/javascript' src="./shadowbox.js"></script>
<script type='text/javascript'>
$(function(){
	shadowbox({
		background: 'url(./tile.png)'
		opacity: .7,
	});
});
</script>
<body>
    <div class='teapot-container'
        shadowbox="open-me">
        <h4>I'm a little teapot</h4>
    </div>
    <div class='peekaboo-container'
        shadowbox="open-me-too">
        <h4>Peekaboo</h4>
    </div>

    <button type='button'
        id='open-button'
        class='btn btn-default btn-centered open-me'>Open me</button>

    <button type='button'
        id='open-button'
        class='btn btn-default btn-centered open-me-too'>Open me too</button>
</body>
